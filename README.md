# Python Mini Projects

A collection of beginner-friendly Python projects demonstrating clean code practices and fundamental programming concepts.

---

## 📌 Project Description

This repository contains small yet practical Python projects designed to strengthen core programming skills such as logic building, file handling, and user interaction.

Each project is simple, well-structured, and easy to understand, making it ideal for beginners and portfolio building.

---

## 🚀 Features

- Clean and readable code
- Beginner-friendly logic
- Command-line based interaction
- Practical real-world use cases
- Modular and easy to extend

---

## 🧰 Tech Stack

- Python 3
- Standard Library (os, shutil)

---

## 📂 Projects Included

### 1. Calculator
A simple calculator that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

---

### 2. To-Do List
A command-line based task manager:
- Add tasks
- View tasks
- Delete tasks

---

### 3. File Organizer
Automatically organizes files into folders based on their extensions:
- Groups files by type (e.g., .jpg, .pdf)
- Creates folders automatically
